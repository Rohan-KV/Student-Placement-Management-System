# College Placement Management System

A CLI-based backend system built using Python and SQLite to simulate real-world college placement workflow.

## Features
- Role-based login (Admin & Student)
- Company eligibility filtering
- Application tracking
- Offer management
- Placement analytics

## Tech Stack
- Python
- SQLite
- Hashlib (Password hashing)

## How to Run
1. Clone repository
2. Run main.py
