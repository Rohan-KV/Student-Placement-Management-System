# College Placement Management System

A web-based placement management system built with Flask and SQLite for managing student applications and campus recruitment workflows.

## Features
- Admin and Student authentication
- Admin: manage companies, view students, review applications
- Student: register, view eligible companies, apply, accept offers
- Eligibility filtering by CGPA
- Clean HTML/CSS/JS interface with search filters

## Tech Stack
- Python (Flask)
- SQLite
- HTML / CSS / JavaScript

## How to Run
1. Create a virtual environment (optional) and install Flask: `pip install flask`
2. Run the app: `python app.py`
3. Open `http://127.0.0.1:5000/`
4. If no admin exists, you will be prompted to create the first admin account.
